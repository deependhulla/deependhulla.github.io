CloudDrive Android App privacy policy

CloudDrive is an Webdav client for Android that uses the user’s WebDAV provider to send and retrieve files (example NextCloud Server).

The app requests the following privacy-relevant Android permissions

Read from/write to Storage (Photos/Media/Files)

The storage area is used to save files for offline use or export app settings. It can also be used to import app settings or select files to upload to CloudDrive Service provided for upload/backup purpose.

Sensitive user information is only used to perform the basic functionality of the app, upload and downloading files. CloudDrive App does not automatically collect and send data to the developers of the app or any third party. Sensitive information is also not included in application logs unless this is enabled by the user. OpenMail Android App privacy policy

OpenMail is an email client for Android that uses the user’s email provider to send and retrieve messages.

The app requests the following privacy-relevant Android permissions:

Read/modify your Contacts

The names and email addresses in the user’s address book (contacts) are used to provide suggestions and auto-complete recipients when composing a message.

Read from/write to Storage (Photos/Media/Files)

The storage area is used to save attachments or export app settings. It can also be used to import app settings or select files to attach to a message the user is composing.

Sensitive user information is only used to perform the basic functionality of the app, sending and receiving email. OpenMail App does not automatically collect and send data to the developers of the app or any third party. Sensitive information is also not included in application logs unless this is enabled by the user.

OpenSync Android App privacy policy

When you enter your login data (DAV server URL, user name, password) in OpenSync, the user name and password will be sent to the server you specified and nowhere else.

User name and password are stored using Android’s account manager and not by OpenSync itself. To protect your login data, please consider using full-disk encryption.

Contacts, events and tasks are stored by the respective Android content providers and not by OpenSync itself. User data won’t be sent to any other server than the given CalDAV/CardDAV server. OpenSync App doesn’t collect, process or transmit any statistical data like app or server usage. There are no ads and there’s no tracking.
